📖 Project Overview

The Smart Grievance Redressal System (SGRS) is a web-based application designed to improve grievance handling in educational institutions. It replaces traditional manual systems with a secure, transparent, and efficient digital platform.

The system allows students to submit complaints (even anonymously), while administrators and teachers manage and resolve them through a structured workflow.

🎯 Objectives
Primary Objectives
Provide a secure grievance management system
Enable anonymous complaint submission
Reduce complaint resolution time
Ensure transparency and accountability
Secondary Objectives
Real-time updates and notifications
Better communication between users
Generate analytical reports
User-friendly interface
🚀 Key Features
🕵️ Anonymous complaint submission
🆔 Unique tracking ID for each complaint
📊 Role-based dashboards (Student, Teacher, Admin)
🔐 Secure authentication using JWT
📡 Real-time complaint tracking
📈 Analytical insights for administrators
🏗️ System Architecture

The system follows a multi-layer architecture:

Client Layer → User Interface
Application Layer → Backend (Node.js, Express.js)
Business Logic Layer → Complaint processing
Database Layer → MongoDB

🔄 Workflow
Student submits complaint (anonymous option available)
System generates tracking ID
Admin reviews complaint
Complaint assigned to teacher
Teacher updates status
Student tracks complaint
Complaint resolved
